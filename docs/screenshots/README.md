## Screenshot Guidelines

- **Resolution**: 1920x1080 or higher
- **Format**: PNG with transparency where applicable
- **Theme**: Include both light and dark mode versions
- **Content**: Use realistic but non-sensitive sample data
- **Annotations**: Add highlights or callouts for key features (optional)

## How to Capture Screenshots

1. Run the development server: `npm run dev`
2. Navigate to the desired page
3. Use a tool like:
   - **Windows**: Snipping Tool or Win + Shift + S
   - **Mac**: Cmd + Shift + 4
   - **Browser**: DevTools device toolbar for responsive views
4. Save with descriptive filenames
5. Optimize file size with tools like TinyPNG
